Trancoding Lycian

The ucode function transcodes from ASCII to Unicode in the Lycian range.

using Lycian
Lycian.ucode("prNnawatE") 

# output

"𐊓𐊕𐊑𐊏𐊀𐊇𐊀𐊗𐊚"
