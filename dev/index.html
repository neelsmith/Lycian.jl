Lycian.jl Documentation

Lycian.jl supports working with a digital corpus of Lycian texts. It allows you to:

  • validate the orthography of a text, and convert between an ASCII-based orthography and the Lycian range of Unicode.
  • tokenize text in Lycian using the MID Orthography interface for parsing texts into classified tokens
  • analyze the morphology of lexical tokens using the CitableParserBuilder module. Analyses identify lexemes and forms with URN values that can be used to look up entries in a Lycian lexicon, and a Lycian morphological database, respectively.
